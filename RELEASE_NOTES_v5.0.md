# NotesForge v5.0 Release Notes

## Highlights
- New backend architecture with dedicated parser, theme engine, template engine, export service, and security hooks.
- Stable JSON API contract for preview/generate/templates/themes with typed request/response models.
- React + TypeScript strict frontend with split editor/preview workflow and 500ms debounced preview requests.
- AI prompt generator panel with template regeneration flow and Open ChatGPT deep link.
- Theme save support and local persistence of selected theme/template.
- Export support for DOCX/PDF/HTML/MD/TXT with metadata stripping, DOCX read-only option, watermark support, and PDF passwording.
- Offline fallback preview rendering when backend is unavailable.
- Authoritative README, architecture diagrams, screenshot assets, GIF demo, and sample content.
