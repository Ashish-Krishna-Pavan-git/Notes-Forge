ğŸ”§ IF SOMETHING GOES WRONG
===========================

Problem: "Backend Offline" message
Solution:
  1. Make sure you copied all Python files to backend/
  2. Check Python is installed: python --version
  3. Try manual start: cd backend && python backend_server.py

Problem: Frontend won't start
Solution:
  1. Delete node_modules: cd frontend && rmdir /s node_modules
  2. Reinstall: npm install
  3. Try again: npm run dev

Problem: Port already in use
Solution:
  1. Run STOP.bat to kill existing processes
  2. Or manually: netstat -ano | findstr :8000
  3. Then: taskkill /F /PID <the_number>
