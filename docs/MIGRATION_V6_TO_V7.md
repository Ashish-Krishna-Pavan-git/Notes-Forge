# Migration v6 to v7

## Goals
- Preserve existing content and APIs.
- Add v7 capabilities without destructive changes.

## What changed
- Version updated to `7.0.0`.
- Parser adds academic markers and figure/table caption nodes.
- Export pipeline adds image/figure rendering and chapter-aware numbering.
- Theme/template catalogs expanded with required v7 defaults.
- Guide available in-app and at `/guide`.

## What stays compatible
- Existing markers still parse.
- Existing API routes unchanged.
- Existing exports still available.
- Existing entrypoints and deployment shape preserved.
